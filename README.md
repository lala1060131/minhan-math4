- 均一影片一律用「點縮圖開新視窗」，**不要**改成 iframe 就地播放（曾測試會在部分環境播不動）

## 功能頁說明

### 首頁（index.html）
- 側邊欄 9 項：首頁／學習地圖／課程練習／錯題本／我的收藏／學習紀錄／每日簽到／設定（已依需求移除「獎勵商城」）
- 頂部「開始學習」按鈕：自動跳轉到**目前未完成的單元**（讀取 `sm_progress`，找第一個 done<10 的單元；全部完成則回到單元1）
- 每張單元卡右上角有⭐收藏按鈕，點擊寫入 `sm_favorites`
- 統計列（星星／連續天數／滿分單元數）皆讀取真實 localStorage 資料，非假資料

### 單元頁（unit1~10.html）
三分頁結構：
1. **📖 重點整理**：3 個必記重點＋1 個範例拆解
2. **🎬 教學影片**：均一單元總覽連結＋精選影片縮圖卡（詳見下方查證狀態）
3. **💗 開始挑戰**：10 題測驗，兔兔在進度軌跳格子，答對集星星、答錯自動寫入錯題本

### 學習紀錄 / 每日簽到 / 我的收藏 / 學習地圖 / 課程練習 / 設定
皆為獨立頁面，透過共用的 localStorage 資料串連（詳見下節）。

## localStorage 資料結構

| Key | 格式 | 說明 |
|---|---|---|
| `sm_name` | string | 使用者名字（設定頁可改，預設「佳洵」） |
| `sm_stars` | number | 全站累積星星數 |
| `sm_u{N}_best` | number | 單元 N 的歷史最高分（N=1~10） |
| `sm_progress` | `{u1:{done,total,ts}, u2:{...}, ...}` | 各單元完成度，done 為最高分數（0~10） |
| `sm_wrong` | `[{unit,unitName,q,options,answer,chosen,explain,count,ts}, ...]` | 跨單元錯題陣列 |
| `sm_checkin` | `["2026-07-10", ...]` | 簽到日期字串陣列（YYYY-MM-DD） |
| `sm_favorites` | `["u1","u3", ...]` | 收藏的單元 id 陣列 |

新增單元時記得同步：
1. `index.html` 的 `for(const uid of [...])` 進度讀取迴圈
2. `index.html` 單元卡的 onclick 連結與 `id="uNbar/uNnum/uNstar"`
3. `mistakes.html` 的 `UNIT_LINKS` 物件
4. `history.html`、`map.html`、`practice.html` 的 `UNITS` 陣列

## 如何修改

- 想調整某單元的重點整理或題目：直接編輯對應的 `unitN.html`，題庫在 `<script>` 內的 `const QS=[...]` 陣列
- 想改配色：所有頁面共用同一套 CSS 變數命名習慣（`--pink`、`--lavender`、`--mint`... 等），改 `:root{}` 裡的色碼即可
- 想新增第 11 個單元（若翰林版有補充教材）：複製任一 `unitN.html` 當模板，記得同步更新「localStorage 資料結構」段落列出的 4 個檔案
- 修改後直接 `git add / commit / push` 到 `main` 分支，GitHub Pages 會自動重新建置（約 1-2 分鐘生效）
